# Projet 7 - Optimisation d'investissement

## Description

Projet Python realise dans le cadre de la formation OpenClassrooms. L'objectif est de selectionner la meilleure combinaison d'actions a acheter en respectant un budget maximum, puis de comparer une approche brute force avec une approche optimisee.

## Competences mises en pratique

- Lecture et nettoyage de donnees depuis des fichiers CSV
- Recherche de combinaison optimale sous contrainte de budget
- Comparaison entre algorithme brute force et algorithme optimise
- Manipulation de structures de donnees Python
- Gestion des donnees invalides ou dupliquees

## Fichiers principaux

- `bruteforce.py` : solution exhaustive basee sur toutes les combinaisons possibles
- `optimized.py` : premiere version optimisee
- `Backtest.py` : version adaptee aux datasets plus volumineux avec gestion d'anomalies

## Lancer le projet

```bash
python bruteforce.py
python optimized.py
python Backtest.py
```

Les fichiers CSV attendus doivent etre places dans le dossier indique par la variable `CSV_FILE` de chaque script.

## Objectif portfolio

Ce projet montre ma capacite a analyser un probleme algorithmique, a ameliorer les performances d'une solution et a manipuler des donnees avec Python.
